01. Initially push all the project files to a github repository. Below are the project files
	•  server code - app.py
	•  pkl files - timemodel.pkl , timemodel_cols.pkl
	•  Procfile
	•  requirements.txt

02. Then Sign in to the heroku and create a new app. Specify both name and region.

03. Click on the deploy tab, click on "connect to github", specify the repository name and connect to it. 

04. Then scroll down and click on deploy branch. Select the appropriate branch if you have more than one branch in the repository.

05. Finally use the URL generated by heroku to connect both mobile application and Machine learning result.
